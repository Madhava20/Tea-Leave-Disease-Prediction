Models Employed for the project

Transfer Learning Models.

VGG-16

VGG-16, or the Visual Geometry Group's 16-layer model, is a deep convolutional neural network architecture used for image classification tasks. It's known for its simplicity and consists of 16 layers with trainable parameters, including 13 convolutional layers and 3 fully connected layers. VGG-16 gained popularity for its effectiveness in various computer vision tasks, although its large number of parameters makes it computationally expensive.

Inception V3

Inception V3 is a deep convolutional neural network architecture designed for image classification and object detection tasks. It's part of the Inception family developed by Google's research team. Inception V3 is known for its efficient use of computational resources and incorporates inception modules, which use multiple filter sizes in parallel to capture features at different scales. This helps improve the network's ability to recognize patterns and objects in images. Overall, Inception V3 balances accuracy and computational efficiency.

Xception

Xception, short for "Extreme Inception," is a deep convolutional neural network architecture. It's closely related to the Inception architecture but introduces a novel approach called depthwise separable convolutions. In Xception, each convolutional operation is split into a depthwise convolution and a pointwise convolution, which helps reduce the number of parameters while maintaining expressive power. This design makes Xception more computationally efficient compared to traditional convolutional architectures. It's particularly useful for image classification tasks and has shown strong performance in various computer vision applications.
